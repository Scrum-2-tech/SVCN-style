Conexión e Integración Externa

La API de SVCN Style permite a integradores y aliados comerciales consultar el catálogo oficial de prendas oversized en tiempo real. Todas las respuestas se devuelven en formato application/json codificado en UTF-8.

GET /api/productos/

Obtiene la lista completa de prendas de vestir disponibles en el sistema.

Parámetros Query Opcionales
Parámetro Tipo Descripción Ejemplo
categoria string Filtra los productos por el nombre de la categoría. ?categoria=Hoodies

🚀 Código de Ejemplo para Integración (JavaScript)

        
Copia el siguiente fragmento de código para consultar nuestra API mediante fetch e inyectarlo en cualquier sitio web externo:

        
// Función para consumir el catálogo de SVCN Style
function cargarCatalogoSVCN() {
    fetch('{{ ejemplo_url }}')
        .then(response => response.json())
        .then(data => {
            console.log("Catálogo de Ropa Cargado:", data);
            // Renderiza en tu sitio web externo
            data.forEach(prenda => {
                console.log(`${prenda.nombre} - Price: $${prenda.precio}`);
            });
        })
        .catch(error => console.error("Error al conectar con SVCN API:", error));
}
